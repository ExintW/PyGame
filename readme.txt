### Move 移动

“mov x y”

- x = 左右位移
- y = 上下位移
e.g. “mov 1 -1” ⇒ 往右下 走一格
e.g. “mov 2 0” ⇒ 往右走两格

---------------------------------------------------------------------------------

### Attack 攻击

“atk <target>”

- target = 敌方角色图标
e.g. “atk A” ⇒ 攻击敌方弓箭手 (A = Archer)

---------------------------------------------------------------------------------

### Ability 技能

“abl <num> <player> <symbol>” → 既可以给敌方也可以给友方施法的用这个
“abl <num>” → 只能给自己施法的用这个（Buff类技能）
“abl <num> <symbol>” → 只能给敌方或者只能给友方施法的用这个（攻击类或治疗类）

- <num> ⇒ 第几个技能
e.g. abl 1 W ⇒ 向战士（己方或敌方）释放自己的第一个技能
e.g. abl 2 ⇒ 向自己释放自己的第二个技能

---------------------------------------------------------------------------------

### Signiture Ability 核心技能

“sig”

- 使用“sig”进入施法前摇
- 等待施法前摇结束就可以用技能
- 施法前摇期间无法控制角色

---------------------------------------------------------------------------------

### End 结束回合

“end”

- 跳过回合

### 角色技能 ###

### Archer
1. Power Shot: 对敌人造成伤害并且击退一格 （击退方向取决于角色面对的方向）
2. Precision: 对自己施加攻击距离+1 buff （攻击两次后结束）
被动技能：开局自带一次移动距离+1 buff
核心技能：Ashe Arrow: 施法前摇结束后选择一个方向放艾希大招（弹道技能）
    - 可以斜着放
    - 每走一格加一伤害，每走两格加一眩晕回合
    - 选择施法方向：1 0 = 像右放，1 -1 = 像右下放

### Warrior
1. Charge: 对自己释放一次伤害+1 buff和一次移动距离+1 Buff
被动技能：开局自带一次攻击伤害+1 Buff

### Mage
1. Ignite: 对敌人施加3回合的点燃
被动技能：开局自带一次攻击距离+1 buff （技能同样会+1）
核心技能： Blaze：施法前摇结束后像指定地点释放3x3的地图燃烧

### Knight
1. Fortify: 对自己施加持续两次的防御+1 Buff
2. Bash: 像敌方目标施加1回合眩晕

### Healer 
1. Heal: 给友方单位加2血量

### 游戏机制 ###

- 斜着移动需要2个移动距离
- 斜着攻击只需要1个攻击距离
- 施法范围 = 角色攻击范围
- 施法范围计算 = 攻击范围计算
- 如果移动到的位置有角色的话会触发踩头机制：
    - 移动几格到那个位置就会往后跳几格（机制跟提莫蘑菇一摸一样）
    - 如果跳完又踩到角色的话会继续跳
    - e.g. ...W..A... => 这个时候W向右移动两格踩到A就会再像右跳两格

###异常机制
- 点燃和眩晕属于异常机制
- 异常机制是挂在角色身上
- 每次轮到这个角色的回合的时候结算一次伤害 （同时-1持续回合）

###地图特效机制
- 地图燃烧属于地图特效机制
- 取决于哪些角色站在有地图特效的格子上
- 每次换玩家回合的时候结算一次伤害 （跟异常不一样）
    - 比如说p1的法师释放了地图特效，等所有p1角色行动完之后就会结算一次伤害
    - 然后等p2所有角色行动完之后又会结算一次伤害
- 持续时间会在每次结算伤害的时候-1

###弹道机制
- 艾希大招属于弹道机制
- 弹道会根据他的速度每个回合移动
- 弹道会在每次换玩家回合的时候结算伤害或者移动 （跟地图特效一样）
- 当弹道跟玩家重叠的时候会只显示玩家（但是弹道还是在的）
- 弹道技能刚释放的时候是从释放玩家的旁边一格开始
    - 比如说A往右（1 0）释放了艾希大招
    - 那弓箭就会从A的右边一格开始走
    - 艾希大招伤害也从那个地方开始计算
